Entendendo Bancos de Dados

Introdução

Diferença de dados e informação:

Todo dado é um objeto onde pode ser tirado uma informação. A informação não existe sem um dado que gerou ela, pois a informação é um dado interpretado e estudado.

Então pode se dizer que a diferença entre um dado e uma informação é basicamente pensar que uma data de aniversário é apenas um dado, logo quando se interpreta e busca meios de entender esse dado, essa data acaba se transformando na idade de uma pessoa, ou seja, uma informação.
